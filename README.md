CraDet is an IoT-based real-time accident detection and emergency response application built using Kotlin in Android Studio. The app uses smartphone sensors such as the accelerometer and gyroscope to monitor sudden impacts, abnormal motion, and possible crash events in real time.
The system calculates live G-force values, analyzes motion patterns, and detects potential accidents using multi-signal sensor logic. In emergency scenarios, CraDet is designed to trigger alerts, track location, and support rapid response workflows.
Key features include:
Real-time accelerometer and gyroscope monitoring
Live G-force calculation
Crash detection logic
Modern sensor dashboard UI
Emergency alert workflow foundation
IoT and wearable integration ready architecture
CraDet is designed as a lightweight, hackathon-friendly prototype focused on practical accident detection and smart emergency assistance.
